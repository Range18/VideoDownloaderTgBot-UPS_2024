## Description

Telegram bot to download videos.

## Installation

```bash
$ yarn install
```

## Enviroment

You have to specify API_ID and API_HASH in .env file. To see more information [check this.](https://core.telegram.org/api/obtaining_api_id)


## Running the app

```bash
# build 
$ yarn run build

# run
$ yarn run start

# watch mode
$ yarn run start:dev
```
